# Eren Gülmez — Claude Code plugin marketplace

One `/plugin marketplace add` for all of my Claude Code plugins.

```
/plugin marketplace add gulmezeren2-byte/claude-marketplace
/plugin install erp-report-engine@gulmezeren
/plugin install industrial-engineering-ai-skills@gulmezeren
```

## Plugins

- **[erp-report-engine](https://github.com/gulmezeren2-byte/erp-report-engine)** — a *provably* read-only SQL layer for AI agents over the database behind an ERP (Logo Tiger, Netsis, Mikro). A guarded MCP server (the agent queries canonical entities through a statement-checking read-only guard, measured against a public 28-attack benchmark), the safe-query skills, and zero-setup `/trust-benchmark` and `/erp-schema` commands.
- **[industrial-engineering-ai-skills](https://github.com/gulmezeren2-byte/industrial-engineering-ai-skills)** — an operations-analysis method pack: OTIF audits, forecast-accuracy reviews, ABC-XYZ segmentation, safety-stock stress tests, Pareto discipline and reporting contracts — with role agents and always-on data-hygiene rules.

More at [github.com/gulmezeren2-byte](https://github.com/gulmezeren2-byte).
